ClassName

ClassName is the name of the class to be created in this program. Note that if this class is public, it reflects the name of the file in which this class is defined.

Java class names and each new word in the variable name should start with an uppercase letter. Underscores should not be used. Good class names include:

Person
MyClassLibrary
NumberFormatter

Examples of names that do not follow the Java standard are:

person
My_Class_Library
numberFormatter

Names that would be invalid in Java are any names that do not follow the Java identifier syntax given in module 1, for example:

2much
kukla,fran&ollie
