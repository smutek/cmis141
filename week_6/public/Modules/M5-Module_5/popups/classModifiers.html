ClassModifiers

A number of modifiers—final, abstract, and public—can be used with a class. The modifiers final and abstract, however, have meaning only when dealing with inheritance. Therefore, the only modifier we will consider is public.

The public modifier indicates that this class can be used anywhere in the program. The public modifier also indicates this class must be in a file named to match the ClassName. For example, if this class is named MyMethods, and the public modifier is used, the class must be in a file named MyMethods.java. Note that because the public class name implies the file name, only one public class can be in any source file.

Every file in this course should contain one class that uses the public modifier. Though allowed by Java with restrictions, for this course, any class not declared public should not be used outside of the file in which it is defined.
