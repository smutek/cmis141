Welcome to week 6 in CMIS 141. Once again, there are multiple activities for you to participate. To be successful, it is critical that you read and complete all of these activities. Consider using this document as checklist of items to complete and roadmap for learning the materials.

The activities for this week include:

1. Content - This module provides the technical details supporting your learning. You should carefully read these modules trying the activities and exercises embedded in the materials:

  a. Thinking in Objects - This sub-module describes how to use constructors, the this reference, methods and static methods.

  b. Using an IDE - This sub-module describes how to download and install Netbeans.

2.  Hands-On Lab - After reading all the competencies, this hands-on lab will walk you through each major concept demonstrating the functionality in Java. There are two hands-on exercises including:

a. Compare and contrast class and local variables

b. Use the this reference

c. Create overloaded constructors

d. Use an IDE to compile and run an application consisting of more than one class

Each of these exercises includes a demonstration using Java code as well as exercises for you to complete to become comfortable using the concepts covered in the Competencies module.

5. Code Examples - This module provides the actual Java code used in the hand-on lab for this week. These may save you some time as you can use the code as opposed to copying and pasting from the hands-on lab document.

6.  Code Solutions - This module provides possible solutions to the exercises in the hands-on lab for this week. You should try to solve each exercise yourself first before looking at the solution. However, if you get stuck, these should help get you passed the issue. Keep in mind, there are hundreds of possible solutions to these exercises. These solutions represent just one possible solution.

7. Assignments - This week you need to respond to a discussion item on creating Java class with constructors, methods and data fields. There is also a homework assignment with programming projects to be completed and submitted.

8. Questions - Post any questions in the "Ask the Professor" discussions area.
