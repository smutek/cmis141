Welcome to week 7 in CMIS 141. Once again, there are multiple activities for you to participate. To be successful, it is critical that you read and complete all of these activities. Consider using this document as checklist of items to complete and roadmap for learning the materials.

The activities for this week include:

1. Content - This module provides the technical details supporting your learning. You should carefully read these modules trying the activities and exercises embedded in the materials:

  a. More on Strings - This sub-module describes how to use the String, StringBuffer and StringBuilder classes in Java.

  b. Dates in Java - This sub-module use classes within the java.time package found in Java 8 and higher.

c. Command Line Arguments – This sub-modules describes how to use command line arguments to send data into a Java application.

2.  Hands-On Lab - After reading all the competencies, this hands-on lab will walk you through each major concept demonstrating the functionality in Java. There are three hands-on exercises including:

a. Compare, contrast and use the String, StringBuffer and StringBuilder classes

b. Use classes in the java.time package

c. Use command line arguments in Java

Each of these exercises includes a demonstration using Java code as well as exercises for you to complete to become comfortable using the concepts covered in the Competencies module.

5. Code Examples - This module provides the actual Java code used in the hand-on lab for this week. These may save you some time as you can use the code as opposed to copying and pasting from the hands-on lab document.

6.  Code Solutions - This module provides possible solutions to the exercises in the hands-on lab for this week. You should try to solve each exercise yourself first before looking at the solution. However, if you get stuck, these should help get you passed the issue. Keep in mind, there are hundreds of possible solutions to these exercises. These solutions represent just one possible solution.

7. Assignments - This week you need to respond to a discussion item on creating a Java application to use command line arguments to generate random numbers. There is also a homework assignment with programming projects to be completed and submitted.

8. Questions - Post any questions in the "Ask the Professor" discussions area.
