Module 5: Objects and Classes

In module 3, we introduced methods and described main as a method. In fact, we began calling it the main method. If we take a closer look at it, we will find that it is just like the other methods with parameters. The parameter used in main is args and it is of the String[] type. As in other methods, it is possible to pass arguments to the main method as shown in the following two classes.

public class Sample1 {
   public static void main(String[] args){
      String[] stringThing = {"Hello", "World!"};
      Sample2.main(stringThing);
   }
}

public class Sample2 {
   public static void main(String[] args){
      for (int i = 0; i < args.length; i++){
          System.out.println(args[i]);
      }
   }
}

To run this program, you would type:

javaSample1

and your output would be:

Hello
World!

Notice that Sample1 sends the stringThing array to Sample2, which then iterates through the string array, printing out each item.

It is also possible to pass strings to a main method when using the command line. Suppose we have created a program called BigMain. In the command line, it is possible to enter the following code:

java BigMain Well Hello World!

Doing so will result in the three strings—Well, Hello, and World!— being passed into the main method. To pass a string containing spaces, the string must be enclosed in quotes. The following is an example with four strings—Well, Hello, World!, and this is it. Note that the fourth string—this is it—is enclosed in quotes to preserve the spaces.

java BigMain Well Hello World! "this is it"

It is possible to pass numbers, too, but they will be treated as strings. Earlier in this module, we demonstrated the way to convert a string to an integer or a double using a wrapper class:

numInt = Integer.parseInt("89");

Here is an example that uses the command line to pass a number as a string:

java Sample3 89

into the following program:

public class Sample3 {
   public static void main(String[] args){
      int numInt = 0;
      numInt = Integer.parseInt(args[0]);
      System.out.println(numInt); OUTPUT
   }
}
